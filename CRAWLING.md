# Crawling Policy

## Repository Scope

This repository is hosted on GitHub. GitHub repository pages do not provide a repository-local `robots.txt` file controlled from this codebase.

That means this file is a project policy statement, not a technical enforcement mechanism.

## Expectations

- Respect GitHub rate limits and platform terms when accessing repository content.
- Preserve attribution when reusing code or documentation under the repository license.
- Do not represent generated summaries or derived datasets as official project output.

## AI And Bulk Reuse

The project's AI behavior is documented in `AI_POLICY.md`.

Any future website or docs site for this project should publish a real `robots.txt` file and machine-readable crawl rules there.

## Citation

If a project site is added later, crawling controls should follow the Robots Exclusion Protocol as standardized in RFC 9309:

- <https://www.rfc-editor.org/rfc/rfc9309>
