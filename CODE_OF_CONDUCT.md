# Code Of Conduct

## Our Standard

This project expects contributors to:

- communicate clearly and respectfully
- focus on technical substance
- assume good intent
- welcome constructive review
- avoid harassment, personal attacks, or discriminatory behavior

## Unacceptable Behavior

- insults or intimidation
- sexualized language or imagery
- doxxing or threats
- disruptive bad-faith behavior
- repeated off-topic escalation after moderation

## Enforcement

Project maintainers may edit, remove, or close content that violates this code of conduct.

Repeated or severe violations may result in temporary or permanent removal from project spaces.
